# Guias – Construção de Componentes

Toda a documentação referente à **construção de componentes** do Jellyfish Design System fica neste diretório: **`packages/ui/guides/`**.

**Regra obrigatória (Cursor)**: Sempre que for criar um novo componente, **leia todos os arquivos desta pasta** antes de implementar. Estes guias são a fonte única de verdade para estrutura, JFV, tokens, utilities, processo e checklists.

## Documentos disponíveis

| Documento | Descrição |
|-----------|-----------|
| [COMPONENT_GUIDELINES.md](./COMPONENT_GUIDELINES.md) | Estrutura de componentes, JFV (variants no pacote ui), tokens (vars no pacote tokens), templates e checklists. |
| [UTILITY_CLASSES_GUIDE.md](./UTILITY_CLASSES_GUIDE.md) | Uso de utility classes (`@jellyfish/tokens`), incluindo a regra de dimensões quadradas (`.size-*`) e referência ao `utilities.css`. |

Novos guias (acessibilidade, composição, testes, etc.) devem ser adicionados aqui e referenciados neste README.
